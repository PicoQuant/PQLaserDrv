Building
--------

Compiling was successful tested with

 * Xamarin 5.9 (Windows 7 (64-bit))
 * Mono 3.2.8 (Ubuntu 14.04.3 LTS)


Running
-------

 1. check if your runtime is 32- or 64-bit
 2. copy 'Sepia2_Lib.dll' with the same architecture (32- or 64-bit) 
    that your runtime has into i.e. the working directory or another
    standard search path
 3. run the executable
